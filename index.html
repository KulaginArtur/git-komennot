Git-komennot

Komenot, jotka tehdään alussa

  • git init
  • git remote add origin https://github.com/käyttäjä/repon-nimi.git
  • git add .
  • git commit -m 'tekstiä'
  • Ensimmäisellä kerralla: git push -u origin master

Yleisimmät komenntot

  • git add .
  • git commit -m 'tekstiä'
  • git push

Muista tehdä README.md ja .gitignore
